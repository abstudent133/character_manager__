# Enhanced RPG Character Manager
***

This project is a Python-based RPG Character Manager that allows users to create, manage, visualize, and analyze characters. The program uses libraries such as Pandas, Matplotlib, and Faker to generate characters, display attribute graphs, and perform statistical analysis on character data. It is designed to run in a Python terminal and provides a menu-driven interface for easy navigation.

## How to use
***
1. Run the program by executing the main file:
python main.py

2. Use the menu to navigate through options such as:
- Creating random or manual characters
- Viewing and visualizing character stats
- Running statistical analysis
- Saving and loading character data

## Requirements / Installation
***
Before running this program, make sure you have Python installed.

Install the required libraries using the following commands in a new terminal:

pip install faker
pip install matplotlib
pip install pandas
pip install numpy


## Details on project features
***
- Character Creation  
  Users can create characters manually or generate them randomly using the Faker library.

- Data Visualization  
  Displays character attributes using bar graphs created with Matplotlib.

- Character Comparison  
  Allows comparison of two characters by displaying their stats side by side.

- Statistical Analysis  
  Uses Pandas to calculate mean, median, maximum, and minimum values for character attributes.

- CSV Data Management  
  Supports saving character data to CSV files and loading it back into the program.

- Menu System  
  Provides a structured terminal-based interface for accessing all features.
